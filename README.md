## LANケーブルの割り当て
### 略語
e:エレベータ  
r:ラダー  
### LANケーブル left
緑白:e up  
緑:e neutral  
オ白:e down  
青:r left  
青白:r neutral  
オ:r right  
茶白:none  
茶:GND  

### LANケーブル right
緑白:e servo  
緑:r servo  
オ白:none  
青:none  
青白:5v  
オ:5v  
茶白:GND  
茶:GND
