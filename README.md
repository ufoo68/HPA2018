# HPA2018 (操舵関係)
## ハードウェア構成
- Arduino nano
- LANコネクタモジュール
- タクトスイッチ
- サーボモータ
- ジョイスティック
## LANケーブルの割り当て
### 略語
e : エレベータ  
r : ラダー  
### LANケーブル left
緑白 : r left  
緑 : r neutral  
橙白 : r right  
青 : e down  
青白 : e neutral  
橙 : e up  
茶白 : none  
茶 : GND  

### LANケーブル right
緑白 : e servo  
緑 : r servo  
橙白 : none  
青 : none  
青白 : 5v  
橙 : 5v  
茶白 : GND  
茶 : GND
